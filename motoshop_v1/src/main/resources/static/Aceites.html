Tipos de Aceite

Aceite para trasmision

El aceite de transmisión es un lubricante encargado de evitar el rozamiento entre el sistema de transmisión y el chasis. Así pues, sin este lubricante las piezas acabarían desgastadas antes de tiempo y podrían generarse averías que afectarían a los elementos que conforman la transmisión del automóvil.

Aceite sintetico 4T

El aceite de motor sintético es el resultado de un proceso de ingeniería química. Las moléculas del aceite sintético tienen una forma más uniforme y contienen menos impurezas que las moléculas de los aceites convencionales.

Aceite para motor 2T

El aceite de motor de 2 tiempos es un lubricante especial diseñado para este tipo de motores, que combina la función de lubricación y enriquecimiento de combustible. Normalmente, este aceite se mezcla directamente con la gasolina y se quema en el motor, proporcionando lubricación y protección contra el desgaste.
